# Portfolio de Mamdou Cheikh Fall

Un portfolio web moderne et responsive créé avec HTML5, CSS3, TailwindCSS et JavaScript.

## 🚀 Fonctionnalités

- **Design moderne** avec TailwindCSS
- **Fully responsive** pour tous les appareils
- **Navigation fluide** avec scroll smooth
- **Animations** au scroll
- **Formulaire de contact** interactif
- **Menu mobile** intuitif

## 📁 Structure

```
portfolio/
├── index.html      # Page principale
├── script.js       # Logique JavaScript
├── README.md       # Documentation
└── assets/         # Images et ressources (à créer)
```

## 🛠 Technologies utilisées

- **HTML5** - Structure sémantique
- **TailwindCSS** - Framework CSS moderne
- **JavaScript** - Interactivité et animations
- **Font Awesome** - Icônes

## 🎨 Personnalisation

### Modifier les informations personnelles

1. Ouvrir `index.html`
2. Modifier les sections suivantes :
   - **Hero section** : Nom et titre
   - **Projets** : Ajouter vos projets réels
   - **Compétences** : Adapter à vos compétences
   - **Contact** : Mettre vos vrais coordonnées

### Ajouter vos projets

Dans la section projets, remplacez les cartes d'exemple par vos vrais projets :

```html
<div class="card-hover bg-white rounded-lg shadow-lg overflow-hidden">
    <div class="h-48 bg-gradient-to-br from-purple-400 to-pink-400"></div>
    <div class="p-6">
        <h3 class="text-xl font-semibold mb-2">Nom du projet</h3>
        <p class="text-gray-600 mb-4">Description du projet</p>
        <div class="flex flex-wrap gap-2 mb-4">
            <span class="px-3 py-1 bg-purple-100 text-purple-600 rounded-full text-sm">Techno</span>
        </div>
        <a href="https://votre-projet.com" class="text-purple-600 hover:text-purple-700 font-medium">
            Voir le projet <i class="fas fa-arrow-right ml-1"></i>
        </a>
    </div>
</div>
```

## 📱 Déploiement

### Option 1 : GitHub Pages

1. Créer un repository GitHub
2. Uploader les fichiers
3. Activer GitHub Pages dans les settings
4. Choisir la branche `main` et le dossier `/`

### Option 2 : Netlify

1. Créer un compte Netlify
2. Glisser-déposer le dossier du projet
3. Votre site est en ligne !

### Option 3 : Vercel

1. Installer Vercel CLI : `npm i -g vercel`
2. Lancer : `vercel`
3. Suivre les instructions
